# VITamin MOM

AI-Powered Healthcare Companion

## Problem

People often delay seeking healthcare due to:
- Fear of diagnosis
- Financial concerns
- Social stigma
- Uncertainty about symptoms

## Solution

VITamin MOM provides:

- Symptom Investigation
- Barrier Detection
- Mother-like Guidance
- Financial Support Suggestions

## Future Scope
